This file defines the `CanvasRenderer` class, which is a canvas-based renderer for tree visualization. It is responsible for rendering the nodes and edges of a graph, as well as for handling user interactions such as panning, zooming, and selecting nodes.

The `CanvasRenderer` class has the following key responsibilities:

*   **Rendering**: It provides methods for rendering the nodes and edges of a graph. It supports a variety of node shapes, such as circles, rectangles, trapezoids, ellipses, diamonds, and hexagons. It also supports a variety of edge styles, such as straight lines, curved lines, dashed lines, and dotted lines.
*   **Interaction**: It provides methods for handling user interactions, such as panning, zooming, and selecting nodes. It also supports touch events and keyboard events.
*   **Styling**: It provides a variety of styling options that can be used to customize the appearance of the graph.
*   **Overlays**: It provides a method for rendering UI overlays, such as a zoom indicator, a node count, and information about the selected node.
